## Tarea
Función: `is_palindrome(s: str) -> bool`

Requisitos:
- Retorna `True` si `s` es palíndromo.
- Ignora mayúsculas/minúsculas.
- Ignora caracteres que no sean alfanuméricos.
- Si `s` es `None`, retorna `False`.

## Restricciones de salida
- Responde con **solo** un bloque de código Python.
- El código debe definir la función `is_palindrome`.

## Casos de ejemplo
- "Anita lava la tina" debe retornar True
- "abc" debe retornar False
- "A man, a plan, a canal: Panama" debe retornar True
